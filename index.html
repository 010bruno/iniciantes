front-end?

Um guia para iniciantes

Nomenclatura

Quando codificamos algum site ou serviço para internet, nos referimos a dois tipos de linguagem: a linguagem back-end e a front-end. O dev back-end (ou server-side), é o responsável em programar as funcionalidades do site/serviço escrevendo código em linguagens como PHP, Ruby, Python, Java, etc.

O dev front-end (ou client-side) é responsável por implementar o design do layout usando códigos de HTML, CSS e Javascript.

Nomenclatura

Por exemplo: se criarmos um script em linguagem back-end (php, asp, java, ruby etc) que apenas calcula a soma de 2 + 2, será o SERVIDOR (ou back, o server) que calculará este resultado. Se fizermos esse cálculo em alguma linguagem front-end, como o Javascript, quem calcula essa conta é o BROWSER do usuário. Por isso o termo client ou front.

Cálculo no Servidor = Back-end ou Server-side

Cálculo no Browser = Client-side ou Front-end
