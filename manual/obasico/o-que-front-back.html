O que é front-end e back-end?

Diferenças entre linguagem front-end e linguagem back-end

FrontEnd e BackEnd são termos relativamente novos. Surgiram basicamente, da necessidade de separar duas grandes camadas de desenvolvimento: o que o usuário vê diretamente, e o que mesmo que ele não veja, faz com que tudo funcione.

Assim, os profissionais que trabalham na interface do usuário, são chamados de Desenvolvedores FrontEnd, e aqueles que trabalham no core da aplicação são os Desenvolvedores BackEnd.

Linguagens FrontEnd

  • HTML (xHTML, HTML4, HTML5...)
  • CSS (CSS2.1, CSS3...)
  • JavaScript Client Side

Linguagens BackEnd

  • PHP
  • ASP
  • Java
  • Ruby
  • Python
  • NodeJS
