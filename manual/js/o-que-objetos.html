O que são Objetos?

Programação orientada a objetos

Em JavaScript, qualquer tipo de dado exceto números, strings, booleanos, null e undefined, são objetos. A linguagem possui também dois tipos de "objetos especiais" que são: Arrays e Funções.

Um objeto é como uma coleção de atributos, ou propriedades. Cada propriedade recebe um nome e armazena um valor que pode ser recuperado em outras partes do programa. O nome de uma propriedade pode ser qualquer string, e o valor que ela armazena pode ser qualquer tipo primitivo (exceto undefined) ou até mesmo um outro objeto. Quando atribuimos uma função ao valor de uma propriedade, obtemos o que é chamado de método.

Além de mapear valores, os objetos também possuem um poderoso recurso chamado herança prototípica, que permite que um objeto herde propriedades e métodos de outros objetos. Essa é uma característica muito importante do JavaScript.

Criando objetos

Notação literal

A forma mais fácil e recomendável de se criar objetos é utilizando notação literal. Um objeto literal é composto por um par de chaves "{}", que envolve uma ou mais propriedades. Cada propriedade segue o formato "nome: valor" e devem ser separadas por vírgula. Exemplo:

var aluno = {
	nome: "João",
	idade: 17
};

No código acima criamos um objeto aluno, que possui duas propriedades: nome e idade. Simples assim. É interessante notar que não colocamos uma vírgula após a última propriedade. Fazendo isso, evitamos problemas em algumas implementações da ECMAScript 3.

Função construtora

Outra maneira comum de se criar objetos é utilizando uma função construtora. Você pode usar funções construtoras nativas do JavaScript e em alguns casos criar construtoras personalizadas. Para inicializar um novo objeto com uma função construtora, você deve usar a palavra-chave new.

var aluno = new Object();
aluno.nome = "João";
aluno.idade = 17;

Acessando propriedades

Podemos recuperar os valores armazenados em um objeto, de duas maneiras:

// notação de ponto
aluno.nome // João

// notação de colchetes
aluno["nome"] // João

Se tentarmos recuperar o valor de uma propriedade que não existe em nosso objeto, será retornado o valor undefined.

aluno.email // undefined

Alterando e adicionando propriedades

O valor de uma propriedade pode ser alterado por uma expressão de atribuição:

aluno.nome = "José";

Caso a propriedade referenciada ainda não existir no objeto, ela será criada:

aluno.email = "jose@exemplo.com";

aluno.status = "Aprovado";

aluno.mudaStatus = function(status) {
	this.status = status;
};

Dados por referência

É importante notar que, diferente dos tipos de dados primitivos, os objetos são armazenados por referência, ou seja, eles não são copiados e gravados diretamente em uma variável.

var alunoUm = { nome: "José" };
var alunoDois = alunoUm;

alunoUm.nome = "João";

alunoUm.nome // João
alunoDois.nome // João

No exemplo acima, nós copiamos o objeto alunoUm para a variável alunoDois, mas ao alterar o valor da propriedade nome do objeto alunoUm, essa mudança também afetou o objeto alunoDois. Isso acontece porque os dados do objeto apenas foram passados por referência, na verdade não foi criado um "clone" com suas próprias propriedades e valores.

Deletando propriedades

Você pode deletar uma propriedade de um objeto utilizando o operador delete.

aluno.idade // 17

delete aluno.idade // remove a propriedade

aluno.idade // undefined
