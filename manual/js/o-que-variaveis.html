O que são variáveis?

Crie, manipule e use valores variáveis

Variável é um dos conceitos mais importantes no estudo de programação, independente da plataforma ou linguagem utilizada.

Variáveis são nomes que referenciam a um espaço na memória do computador. Elas são utilizadas para guardar informações que serão usadas em seus programas.

Para elucidar este conceito, imagine que a memória do seu computador é um armário com 100 gavetas e você guarda cada tipo de coisa em uma gaveta diferente.
Se você não for um super humano, como eu, provavelmente você vai querer criar etiquetas com nomes, referenciando o que está guardado em cada uma delas.

Mas por que o nome "Variável", foi dado este nome, por que elas podem ter seus valores alterados durante a execução de um programa. Voltando a ideia das gavetas, você poderia ter em uma delas 2 meias agora e colocar mais 5 depois de um tempo e caso seja um colecionador 200 no futuro.

Criando variáveis

Para criar uma variável no javascript utilizamos o operador '=' operador de atribuição.
Devemos sempre escolher um nome que identifique o tipo de dado que teremos guardado, para facilitar a leitura de nosso código.

Exemplo:

<script>

  var nome = "Gabriel Mendonça";
  var idade = 25;  

</script>

O operador "var" indica que você está declarando uma variável. Embora não seja obrigatório a utilização é interessante pois assim fica claro em que parte do seu código essa variável foi criada.

Note também que no final de cada linha utilizamos o operador ';' ponto e virgula, que identifica onde a linha de código termina. Esse operador embora não seja obrigatório, faz com que o seu código seja mais preciso e claro.

Brincando um pouco!

Vamos brincar um pouco com as variáveis e assim aprender mais sobre elas. Para isso abra o seu editor de texto preferido. Salve o arquivo .html em uma pasta de sua preferência com a estrutura básica e depois abra este arquivo em um navegador.

É interessante que você digite o código ao invés de copiar e colar para fixar melhor.

Exemplo 1 

<script>

  var nome = prompt('Digite seu nome: ');
  alert(nome + ', seja bem vindo!');  

</script>

Neste exemplo, declaramos uma variável nome onde temos a intenção de guardar o nome da pessoa que está acessando o site e utilizamos a função prompt para pedirmos que este usuário digite seu nome.

Logo depois, decidimos apresentar a mensagem de boas vindas a este usuário, utilizando o nome guardado na variável.

Exemplo 2 

<script>
	// calcular idade

	// declara o ano atual para fazer o calculo
	var ano_atual = 2014;

	// pede que o usuário digite o ano em que nasceu
	var ano_nascimento = prompt('Digite o ano em que você nasceu ');

	// faz o calculo para encontrar a idade do usuário e armazena na variável idade
	var idade = ano_atual - ano_nascimento;

	// mostra ao usuário a idade que ele possui
	alert("Você tem " + idade + " anos");

</script>

Neste exemplo vimos coisas novas e novidades sempre são bem vindas. O operador '//' barra barra, é o operador utilizado quando desejamos adicionar um comentário em uma linha. Um comentário é um trecho dentro do seu código que não é executado, ou seja, ele está ali só para que o leitor entenda o que está acontecendo.

Além do operador de uma linha, temos também o de várias linhas, onde podemos por exemplo, colocar um texto explicativo de alguma coisa. Este operador tem a seguinte sintaxe:
/* texto explicativo */

Definindo nomes de variáveis

Quando criamos variáveis temos que levar em consideração algumas regras da linguagem. Estas regras estão listadas abaixo.

  • Variáveis em javascript são "Case-Sensitive".

    Este termo significa que nomes com letras maiusculas são diferentes de nomes com letras minusculas. Logo, para um programa em javascript, Nome é diferente de nome.

  • Caracteres válidos
    • Letras: MAIUSCULAS e minusculas. Exemplo:

      var NOME = "joão";
      var nome = "josé";
      var NoMe = "Thiago";

    • Números: desde que sejam precedidos de uma ou mais letras. Exemplo:

      var camisa9 = "ronaldo";
      var camisa230lateral = "joaozinho";

    • Underline: "_", pode ser utilizado em qualquer posição da variável. Exemplo:

      var _nome = 'Gabriel'
      var segundo_nome = "Mendonça"
      var ultimo_nome_ = ""

Nome reservados pela linguagem

Tipos de variáveis

Quando falamos em tipos de variáveis temos as linguagens chamadas de fortemente tipadas e fracamente tipadas.

Em linguagens fortemente tipadas, definimos o tipo da variável no momento de sua criação. Exemplos de linguagens do tipo: Java, C, C++, entre outras.

Em linguagens fracamente tipadas, não precisamos definir o tipo da variável, ela é tipada automaticamente quando recebe um valor. Exemplos de linguagens do tipo: Python, Ruby, Javascript, entre outras.

Abaixo segue os tipos de variáveis em javascript.

  • Int - Variáveis com valores inteiros. Exemplos:

    var idade = 17;
    var graus = -3;
    var pontos = 0;
    var numero_grande = 2000009283;

  • Float - Variáveis com ponto flutuante ou casas decimais. Exemplos:

    var peso = 32.59345;
    var PI = 3.14;
    var meu_saldo = -1034.32

    Observação: em linguagens de programação utilizamos "." ponto final, para separarmos casas decimais ao invés da ',' vigula.

  • String - Variáveis de texto, normalmente chamada de cadeia de caracteres. Os valores deste tipo é atribuído utilizando-se (") aspas duplas ou (') aspas simples como delimitador. Exemplos:

    var nome = "Gabriel Mendonça";
    var data_nascimento = "17 de Junho de 1988";
    var email = "gabriel@host2.com.br"; var tempo = "20s";

    Observação: Tudo o que declaramos entre os delimitadores (") ou (') será entendido como parte da string mesmo se só tivermos colocado números.

  • Booleanos - Os valores possíveis são "true" ou "false", verdadeiro ou falso. Exemplos:

    var verdadeiro = true;
    var verdadeiro2 = 1;
    var falso1 = false;
    var false2 = 0;
    var falso3 = null

    Observação: Apesar dos valores true e false representarem os valores verdadeiros e falos, podemos utilizar outros valores para essa representação, como mostrado acima.

  • Arrays - Os arrays funcionam como um conjunto de variáveis organizadas por índice, este índice pode ser um valor "inteiro" ou "string". É muito importante entender bem os arrays. Na sessão Arrays você entenderá melhor seu funcionamento.
  • Objetos - Os objetos serão estudados mais a frente, na sessão O que são objetos .
